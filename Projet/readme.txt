Utilisation de l'application:

Pour lancer le projet il suffit de mettre ce dossier Projet à la racine de votre serveur apache.
Les logins et mdp de la base de données sont dans le fichier database.php
Vous pouvez utiliser le login vthion et mdp test pour vous connecter en tant qu'admin.
Et utiliser le login bvozel et mdp test pour vous connecter en tant qu'user.
